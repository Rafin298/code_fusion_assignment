About Country Info App

Country Info App is a simple web application that provides information about countries around the world. The data is sourced from the RestCountries API and presented in an easy-to-use interface.

Features

  • View basic information about all countries
  • Search countries by name
  • View detailed information about each country
  • Discover countries in the same region
  • Find countries that speak the same language

Technologies Used

Django Django REST Framework Bulma CSS JavaScript RestCountries API

Project Assignment

This web application Inclues:

  • Working with external APIs
  • Backend development with Django
  • Creating RESTful APIs
  • Frontend development with HTML, CSS, and JavaScript
  • Data presentation and user interface design
